# SkillForge
MVP for financial literacy and career mobility using JAAT/O*NET data

## Overview

SkillForge is a career development platform that helps individuals identify skill gaps, explore career pathways, and connect with educational and professional opportunities. Built on JAAT and O*NET data standards, SkillForge provides evidence-based career guidance and skills assessment.

## For Partners

SkillForge offers a comprehensive **B2B Partner Program** designed for agencies, universities, and recruiters. Our institutional partners gain access to:

### Agency & Recruiting Solutions
- **Dedicated API access** for candidate skill matching and assessment
- **Custom dashboards** for tracking candidate progress and competencies
- **Bulk resume processing** and skill extraction capabilities
- **Integration support** for existing ATS and HR systems

### University & Educational Institutions
- **Student portfolio management** and skills tracking
- **Program alignment tools** mapped to industry competencies
- **Career services integration** with placement analytics
- **White-label options** for institutional branding

### Key Benefits
- Real-time skills gap analysis across candidate pools
- Evidence-based matching using O*NET frameworks
- Scalable API infrastructure for enterprise needs
- Comprehensive analytics and reporting dashboards
